--- layout: page title: Manifesto tags: [] status: publish type: page published: true meta: _edit_last: "2" _wp_page_template: default _aioseop_description: A minimalist HTML5 ready Wordpress theme, with just a few bells and whistles, heavily based off of the excellent Manifest theme by Jim Barraud. _aioseop_keywords: wordpress,theme,minimalist --- Manifesto is a minimalist Wordpress theme heavily based on the excellent Manifest theme by Jim Barraud. Following that themes highly readable format it introduces:
  • HTML5 compatibility.
  • Support for a single sidebar of widgets.
  • Automatic shrinking of images in the main column to fit the 500 pixel width.
  • Customizable navigation menus for the header and footer.
  • Customizable header image.
  • A "rich" description feature, to allow for HTML elements in the header description.

Download

Version 1.0 - TAR.GZ | ZIP You can also clone it through git from github.

Installation

  1. Unzip the folder under the wp-content/themes folder of your Wordpress installation.
  2. Log in to your Wordpress site and got to Appearance or Themes.
  3. Activate the Manifesto theme and you're done!

Usage

Customizable Header

Like other Wordpress themes with customizable header images, you can upload and crop the image through the Header link under the Appearances menu in your Wordpress dashboard. The image will not replace the text, but will be anchored to the lower right of the header area, right above the sidebar. The page width is 750 pixels, but you can upload an image smaller than that, as I did on my site; my headshot is 225 pixels wide to match the width of the siderbar.

Customizable Menus

Like other Wordpress themes with customizable menus, you can upload and crop the image through the Menus link under the Appearances menu in your Wordpress dashboard. Remember to save your menus as you edit them! If no custom menus are created, Manifesto defaults to displaying links to your Pages.

Rich Description

The standard description field for Wordpress does not currently support rich elements such as links or formatting (this actually makes sense as this description is used not only on your site but in other places where such elements would come out looking strange). Manifesto allows you to enter a "richer" description with such elements in it's Theme Options, under the Appearance Menu of your Wordpress dashboard. If no rich description is entered, it defaults to the standard description entered under the Settings page.
